# VundS Claude Plugins

A plugin marketplace for Claude Code containing VundS team tools and workflows.

## Installation

```bash
# Add this marketplace
/plugin marketplace add vunds/maude-claude-vunds-plugins

# Install individual plugins
/plugin install crm-customer-analysis@maude-claude-vunds-plugins
/plugin install epct-workflow@maude-claude-vunds-plugins
/plugin install vunds-tools@maude-claude-vunds-plugins
```

## Available Plugins

| Plugin | Description | Category |
|--------|-------------|----------|
| **crm-customer-analysis** | CRM customer analysis workflow | Productivity |
| **epct-workflow** | Structured EPCT development approach | Development |
| **vunds-tools** | General VundS productivity tools | Productivity |

## Plugin Details

### crm-customer-analysis

CRM Customer Analysis Command for VundS - analyze customer data and provide insights.

**Commands:**
- `/crm-analyse-customer` - Run customer analysis

### epct-workflow

Structured development approach using the EPCT methodology.

**Commands:**
- `/epct <task>` - Follow structured development approach for a task

### vunds-tools

General productivity tools and skills for the VundS team.
