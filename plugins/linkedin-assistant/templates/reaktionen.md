# Typische Reaktionen (Benno-Stil)

## Dankes-Antworten

### Auf Komplimente zu Posts
```
Danke für das positive Feedback!
```

```
Freut mich, dass der Beitrag hilfreich war.
```

```
Vielen Dank! Das Thema liegt mir am Herzen.
```

### Auf Glückwünsche
```
Vielen Dank für die Glückwünsche!
```

```
Danke! Freut mich sehr.
```

### Auf Vernetzungsanfragen (mit Nachricht)
```
Hallo [Name],

freut mich, dass wir uns vernetzen! Ihr Hintergrund im Bereich [X]
klingt spannend.

Beste Grüße
Benno
```

```
Hallo [Name],

danke für die Vernetzung! Lassen Sie uns gerne in Kontakt bleiben.

Beste Grüße
Benno
```

## Interesse-Bekundungen

### Zu Projekten/Themen
```
Das klingt interessant! Gerne erfahre ich mehr darüber.
```

```
Spannendes Thema. Haben Sie dazu schon Erfahrungen gesammelt?
```

```
Das deckt sich mit meinen Beobachtungen. Wie gehen Sie damit um?
```

### Zu Zusammenarbeit
```
Das könnte passen. Lassen Sie uns dazu kurz telefonieren.
```

```
Interessanter Ansatz. Wann hätten Sie Zeit für einen Austausch?
```

## Höfliche Absagen

### Bei Zeitnot
```
Danke für die Anfrage! Aktuell ist mein Kalender leider sehr voll.
Gerne können wir in [Zeitraum] nochmal sprechen.
```

```
Vielen Dank für Ihr Interesse. Momentan habe ich keine Kapazitäten,
melde mich aber gerne, wenn sich das ändert.
```

### Bei fehlendem Fit
```
Danke für die Nachricht! Das Thema passt aktuell nicht zu meinen
Prioritäten. Ich wünsche Ihnen viel Erfolg!
```

```
Vielen Dank für das Angebot. Das ist derzeit nichts für mich,
aber vielleicht ergibt sich später eine Gelegenheit.
```

### Bei Verkaufsanfragen
```
Danke für die Info. Aktuell kein Bedarf, aber ich melde mich
bei Interesse.
```

## Kommentar-Antworten

### Zustimmung
```
Absolut! Genau das ist der Punkt.
```

```
Sehe ich genauso. Danke für die Ergänzung!
```

### Auf Fragen
```
Gute Frage! Aus meiner Erfahrung...
```

```
Das ist ein wichtiger Aspekt. Kurz gesagt...
```

### Auf konstruktive Kritik
```
Danke für den Input! Da ist was dran.
```

```
Interessante Perspektive. Das werde ich mitnehmen.
```

## Geburtstags-Glückwünsche

```
Herzlichen Glückwunsch zum Geburtstag! Alles Gute!
```

```
Alles Gute zum Geburtstag! Weiterhin viel Erfolg!
```

## Job-Wechsel Gratulationen

```
Herzlichen Glückwunsch zur neuen Position! Viel Erfolg!
```

```
Tolle Neuigkeiten! Alles Gute für die neue Herausforderung!
```

---

## Formatierungs-Regeln

1. **Kurz halten** - Max 3-4 Sätze
2. **Keine Emojis** - Außer explizit gewünscht
3. **Keine Floskeln** - "Ich freue mich riesig" vermeiden
4. **Authentisch** - Nicht übertrieben enthusiastisch
5. **Immer Benno** - Unterschrift bei längeren Nachrichten
