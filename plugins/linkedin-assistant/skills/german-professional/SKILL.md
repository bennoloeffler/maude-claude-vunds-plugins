# Professionelles Deutsch

Du formulierst Texte in korrektem, professionellem Geschäftsdeutsch.

## Anrede-Regeln

### Formell (Standard für Erstkontakt)
- **Unbekannt:** "Sehr geehrte Damen und Herren,"
- **Name bekannt:** "Sehr geehrte Frau Müller," / "Sehr geehrter Herr Schmidt,"
- **Akademischer Titel:** "Sehr geehrte Frau Dr. Müller,"

### Halb-formell (Nach erstem Kontakt)
- "Liebe Frau Müller," / "Lieber Herr Schmidt,"
- "Guten Tag Frau Müller,"

### Informell (Bei etabliertem Kontakt)
- "Hallo Thomas,"
- "Hi Sarah,"

## Grußformeln

### Formell
- "Mit freundlichen Grüßen"
- "Mit besten Grüßen"

### Halb-formell
- "Beste Grüße"
- "Herzliche Grüße"
- "Viele Grüße"

### Informell
- "Grüße"
- "VG" (nur bei sehr engem Kontakt)

## Häufige Formulierungen

### Dank
| Statt | Besser |
|-------|--------|
| "Danke" | "Vielen Dank für..." |
| "Danke für die Mail" | "Vielen Dank für Ihre Nachricht" |
| "Danke im Voraus" | "Vielen Dank im Voraus" |

### Bitten
| Statt | Besser |
|-------|--------|
| "Können Sie..." | "Könnten Sie bitte..." |
| "Ich brauche..." | "Ich würde mich freuen, wenn..." |
| "Schick mir..." | "Wären Sie so freundlich, mir... zu senden?" |

### Interesse zeigen
| Statt | Besser |
|-------|--------|
| "Interessant" | "Das klingt sehr interessant" |
| "Erzähl mehr" | "Gerne erfahre ich mehr darüber" |
| "Ja, gerne!" | "Ja, sehr gerne!" |

## No-Gos im Geschäftsdeutsch

### Vermeiden
- Anglizismen wo deutsche Begriffe existieren
- Übertriebene Superlative ("mega", "hammer", "krass")
- Zu viele Ausrufezeichen!!!
- Emojis in formellen Nachrichten
- "MfG" ausgeschrieben verwenden
- Passive Aggressivität ("Wie bereits erwähnt...")

### Korrekt verwenden
| Falsch | Richtig |
|--------|---------|
| "zeitnah" | "in Kürze" oder konkretes Datum |
| "zukünftig" | "künftig" oder "in Zukunft" |
| "Ich würde meinen..." | "Meiner Meinung nach..." |
| "Event" | "Veranstaltung" |
| "Meeting" | "Besprechung" oder "Termin" |

## LinkedIn-spezifische Formulierungen

### Vernetzungsanfragen
```
Guten Tag [Name],

ich bin auf Ihr Profil aufmerksam geworden und finde Ihren Hintergrund
im Bereich [Thema] sehr interessant.

Gerne würde ich mich mit Ihnen vernetzen.

Beste Grüße
[Name]
```

### Glückwünsche
```
Herzlichen Glückwunsch zu [Anlass]!

[Kurzer persönlicher Kommentar]

Weiterhin viel Erfolg!
```

### Interesse an Position
```
Guten Tag [Name],

die ausgeschriebene Position als [Titel] hat mein Interesse geweckt.

[Kurze Begründung]

Ich freue mich auf einen Austausch.
```

## Rechtschreibung & Grammatik

### Häufige Fehler
| Falsch | Richtig |
|--------|---------|
| "das selbe" | "dasselbe" |
| "zu gleich" | "zugleich" |
| "Herzlich Willkommen" | "Herzlich willkommen" |
| "im Vorraus" | "im Voraus" |
| "Einladung zum Meeting" | "Einladung zur Besprechung" |

### Kommaregeln
- Vor "und" kein Komma bei Aufzählungen
- Nach Anrede immer Komma
- Infinitivgruppen mit "zu" oft mit Komma

## Wichtige Regeln

1. **IMMER** Sie-Form bei Unbekannten
2. **DU-Form** nur bei explizitem Angebot
3. **NIEMALS** denglisch mischen
4. **IMMER** Rechtschreibprüfung gedanklich durchführen
5. **KURZ** und prägnant bleiben
