# Häufige Fehler im Geschäftsdeutsch

## Rechtschreibung & Grammatik

| Falsch | Richtig |
|--------|---------|
| "das selbe" | "dasselbe" |
| "zu gleich" | "zugleich" |
| "Herzlich Willkommen" | "Herzlich willkommen" |
| "im Vorraus" | "im Voraus" |
| "Einladung zum Meeting" | "Einladung zur Besprechung" |

## Anglizismen vermeiden

| Anglizismus | Deutsche Alternative |
|-------------|---------------------|
| "Event" | "Veranstaltung" |
| "Meeting" | "Besprechung" / "Termin" |
| "Update" | "Aktualisierung" |
| "Feedback" | "Rückmeldung" |
| "Call" | "Telefonat" / "Gespräch" |
| "Check" | "Prüfung" |

## Ungenaue Zeitangaben

| Vermeiden | Besser |
|-----------|--------|
| "zeitnah" | "in Kürze" oder konkretes Datum |
| "zukünftig" | "künftig" oder "in Zukunft" |
| "demnächst" | "in den nächsten Tagen" |
| "bald" | "bis Ende der Woche" |

## No-Gos

### Vermeiden
- Übertriebene Superlative ("mega", "hammer", "krass")
- Zu viele Ausrufezeichen!!!
- Emojis in formellen Nachrichten
- "MfG" - immer ausschreiben
- Passive Aggressivität ("Wie bereits erwähnt...")
- Denglisch mischen in einem Satz

### Kommaregeln
- Vor "und" kein Komma bei Aufzählungen
- Nach Anrede immer Komma
- Infinitivgruppen mit "zu" oft mit Komma
