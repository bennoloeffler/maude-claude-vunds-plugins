# LinkedIn-spezifische Formulierungen

## Vernetzungsanfragen

### Standard
```
Guten Tag [Name],

ich bin auf Ihr Profil aufmerksam geworden und finde Ihren Hintergrund
im Bereich [Thema] sehr interessant.

Gerne würde ich mich mit Ihnen vernetzen.

Beste Grüße
[Name]
```

### Mit konkretem Bezug
```
Hallo [Name],

Ihr Beitrag zu [Thema] hat mich angesprochen. Als [eigene Position]
beschäftige ich mich ebenfalls damit.

Lassen Sie uns vernetzen!

Beste Grüße
[Name]
```

## Glückwünsche

### Zum Jobwechsel
```
Herzlichen Glückwunsch zur neuen Position bei [Firma]!

[Kurzer persönlicher Kommentar zum Fit/zur Firma]

Weiterhin viel Erfolg!
```

### Zum Geburtstag
```
Herzlichen Glückwunsch zum Geburtstag!

Alles Gute und weiterhin viel Erfolg!
```

### Zum Jubiläum
```
Herzlichen Glückwunsch zu [X] Jahren bei [Firma]!

Das ist eine beeindruckende Zeit. Weiter so!
```

## Interesse an Position

```
Guten Tag [Name],

die ausgeschriebene Position als [Titel] hat mein Interesse geweckt.

[Kurze Begründung warum es passt]

Ich freue mich auf einen Austausch.

Beste Grüße
[Name]
```

## Häufige Formulierungen

### Dank
| Statt | Besser |
|-------|--------|
| "Danke" | "Vielen Dank für..." |
| "Danke für die Mail" | "Vielen Dank für Ihre Nachricht" |
| "Danke im Voraus" | "Vielen Dank im Voraus" |

### Bitten
| Statt | Besser |
|-------|--------|
| "Können Sie..." | "Könnten Sie bitte..." |
| "Ich brauche..." | "Ich würde mich freuen, wenn..." |
| "Schick mir..." | "Wären Sie so freundlich, mir... zu senden?" |

### Interesse zeigen
| Statt | Besser |
|-------|--------|
| "Interessant" | "Das klingt sehr interessant" |
| "Erzähl mehr" | "Gerne erfahre ich mehr darüber" |
| "Ja, gerne!" | "Ja, sehr gerne!" |
